# ⚡ MediaTool

Все инструменты для создания контента в одном месте.

## 🚀 Возможности

| Инструмент | Описание |
|-----------|----------|
| 🎬 **Видео** | Скачивание с YouTube, Rutube, Instagram → 1080p |
| 📸 **Фото** | Увеличение в 8x + контраст + цветокоррекция |
| 🖼️ **Фон** | Автоматическое удаление фона (люди, животные, предметы) |
| 📱 **Соцсети** | Трендовые размеры + улучшение кожи лица |

## 📋 Установка

```bash
# 1. Перейдите в папку проекта
cd c:\Users\User\media_bot

# 2. Установите зависимости
pip install -r requirements.txt
```

## ▶️ Запуск

```bash
python app.py
```

Откройте в браузере: **http://localhost:5000**

## 📁 Структура проекта

```
media_bot/
├── app.py                  # Сервер (Flask)
├── requirements.txt        # Зависимости
├── templates/
│   └── index.html          # Главная страница
├── static/
│   ├── style.css           # Стили
│   └── app.js              # Фронтенд-логика
└── uploads/                # Загруженные файлы (автоочистка каждые 12ч)
```

## 🔧 Примечания

- **Удаление фона** требует `rembg` и `onnxruntime` — уже в requirements.txt
- Файлы автоматически удаляются через 12 часов
- Максимальный размер загружаемого файла: 10 МБ
